PROJECTS


ideas.

My group midterm project at Skill Distillery, ideas. was the first full-stack project created from scratch, from the creation of the MySQL database to the JSP pages. The user can read ideas and comments, but cannot make any ideas or comments themselves until they create an account or login. With the notable exception of some security issues, our group was able to account for full CRUD functionality, admin privileges, comprehensive sorting logic, and multiple like prevention for a single user.

SEVEN WONDERS

An individual project, Seven Wonders was one of my later fully dynamic web projects. Users are able to not only read files from the database, but create, update, and delete files as well, making this a functional CRUD application. Unlike some of my earlier projects, this was my first project that integrated all of my previously learned technologies, both HTML and CSS on the front-end, and SQL, Gradle, Spring and JPA in the back-end.

US STATE FINDER

One of my later projects at Skill Distillery, US State Finder was one of my early full-stack projects combining database interaction with MySQL with front-end user interface with HTML. The user can either retrieve state data by name or abbreviation, and the program displays information about the state, including Google Maps location for the capital and a link to the state's Wikipedia page.

FILM QUERY

A collaborative group project, Film Query engages in more advanced database retrieval through foreign key relations between different tables. When searching for a film or list of films, not only are the film's details revealed from the film table, but also the cast from the actor table as associated with the actor's foreign key. The user also has the option of adding, updating, or deleting films from the database.

ROMAN NUMERAL TRANSLATOR

My first encounter with regular expressions, Roman Numeral Translator takes a user's numerical input and converts it to a Roman numeral.

BLACKJACK

One of my early advanced Java programs that operates with class and method interaction. It utilizes enums to create the deck of cards which is used in the game of blackjack against the computer. A highscore .txt file can also score the highest score of rounds won.
